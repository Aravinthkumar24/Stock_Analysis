âš™ï¸ Quickstart (Development)
These steps assume you cloned the repository locally.
1.	Create and activate a virtual environment:
python -m venv .venv
source .venv/bin/activate      # macOS / Linux
.venv\Scripts\activate       # Windows
2.	Install dependencies:
pip install -r requirements.txt
3.	Prepare your dataset:
â€¢	Place YAML files inside dataset_yaml/ organized by month folders (e.g. dataset_yaml/2024-01/*.yaml).
â€¢	Each yaml file should contain an array of records with keys like: Ticker or Symbol, date, open, high, low, close, volume.
4.	Run the pipeline to extract CSVs, compute metrics and create plots:
python data_pipeline.py
# optional flags:
# --no-extract   -> skip YAML extraction (use existing csv_output)
# --no-plots     -> skip plot generation
5.	Start the Streamlit dashboard:
streamlit run streamlit_app.py
Open the browser at the URL printed by Streamlit (usually http://localhost:8501).
